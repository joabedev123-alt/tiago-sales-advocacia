# 📁 Projeto: Tiago Sales Advocacia e Consultoria Jurídica

## 📌 Visão Geral do Projeto

Site institucional premium para o escritório **Tiago Sales Advocacia e Consultoria Jurídica**, desenvolvido com React + Vite + TypeScript + TailwindCSS.

- **Framework:** React + Vite
- **Estilização:** TailwindCSS v3
- **Animações:** Framer Motion
- **Ícones:** Lucide React + React Icons
- **Roteamento:** React Router DOM
- **Dev Server:** http://localhost:5173/
